~Picross~

What is Picross?

Picross or Picture Crossword is a type of nonogram or picture logic puzzle (you could almost think of it as a cross between Sudoku and Minesweeper). The game is set up in a pixel grid where you fill in the boxes with specific colors to complete the picture. Next to the cells are the specific rules on where colors (or blank spaces) must go.

How to Play

Let's take a look at an empty 5x5 grid example to learn how to play:

Right off we can see the numbers in the rows and columns

  • Single numbers represent how many consecutive blocks must be filled in. Hint: easy columns and rows are ones with zeros or the maximum amount of boxes in that row or column. For example if the board is 5x5, then it is safe to fill in all 5 squares in that section.

  • Double numbers represent how many blocks are filled in, according to the first number, then at least one black space, then a group of corresponding consecutive spaces based on the second number. For example on a 5x5 grid it is possible to have a 3 and a 1. This would require 3 consecutive spaces to be filled in that row/column then a blank space, then 1 square filled.

  • When a puzzle uses multiple colors we pay attention to the color of the number indicating how many of that specific color are consecutive.

Like in Minesweeper, "marks" can help us know where we can and cannot place colors. Using a mark like an "X" we can place it in somewhere that must remain blank. This is very useful when zeros are in play; as they can eliminate squares easily.

Completing large numbers first can help us to solve the puzzle efficiently. Look to see if we have any large single numbers in any columns or rows.

Next we can fill in lower numbers like we see in the next picture. There are only 3 spaces left after we figured out empty cells.

We can finally finish the puzzle by evaluating our empty and finished columns and rows.
